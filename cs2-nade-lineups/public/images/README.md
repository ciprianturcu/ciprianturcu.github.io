# Map Images

Place your map background images here. The images should be named according to the map name in lowercase.

## Required Images

- `dust2.jpg` - Dust 2 map image
- `mirage.jpg` - Mirage map image
- `inferno.jpg` - Inferno map image
- `nuke.jpg` - Nuke map image
- `overpass.jpg` - Overpass map image
- `vertigo.jpg` - Vertigo map image
- `ancient.jpg` - Ancient map image
- `anubis.jpg` - Anubis map image

## Image Recommendations

- **Aspect Ratio**: 16:9 for best display
- **Resolution**: At least 1920x1080 for crisp quality
- **Format**: JPG or PNG
- **File Size**: Keep under 500KB per image for faster loading

## Finding Images

You can find high-quality CS2 map images from:
- Official CS2 media/screenshots
- Community workshop thumbnails
- CS2 wallpapers and artwork

Make sure you have the rights to use any images you add to this project.
