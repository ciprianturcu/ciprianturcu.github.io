# ciprianturcu.github.io

Personal portfolio and project showcase website.

## Projects

- **[CS2 Nade Lineups](./cs2-tools/)** - Interactive tool for Counter-Strike 2 grenade lineups and strategies

## Development

Each project is contained in its own directory with its own dependencies and build process.

### CS2 Tools

```bash
cd cs2-tools
npm install
npm run dev
```

## Deployment

The site is deployed to GitHub Pages. To deploy:

```bash
cd cs2-tools
npm run deploy
```
